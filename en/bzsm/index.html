The difference between Minecraft and minecraft

To figure out the difference between the two games

Minecraft

Minecraft (that is, the international version and offline version) is a Java game produced by Mojang and was later acquired by Microsoft. However, due to the official EULA agreement of Minecraft, it is not allowed to modify the game body jar and distribute the original file to others. But it allows players to indirectly use mod loaders (Forge loaders, etc.) to dynamically modify the Java bytecode of the game, thus deriving mods that can expand gameplay and have a huge mod community

Netease Minecraft

Minecraft (NetEase Proxy Version) is a game authorized by Microsoft to allow NetEase to act as an agent in mainland China. The official does not allow players to use mods to load into the game without permission. The game client cannot be offline and must be connected to the Internet to play. The NetEase agreement has written Players cannot modify the game without permission.

#####Mod

The essence of Mod is that a group of bigwigs reverse-engineered the Minecraft client to create mod loaders such as Forge to implement indirect dynamic modifications to the minecraft body, ranging from small memory repairs to various large mods that your computer cannot run.

What is Water Shadow

It is a plug-in based on Minecraft Mod. It uses loaders such as Forge to modify key bytecodes of the game (motion nodes, etc.) to achieve functions such as aimbot (self-aiming) or Speed (speed). Mojang will not officially block it because it is legal and open source on github

What is FDPClient

FdpClient is a client based on Water Shadow and added more functions and more ~~bugs
